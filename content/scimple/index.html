Apache SCIMple™

The Apache SCIMple™ is an implementation of SCIM v2.0 specification.

Using Apache SCIMple™ user and group details can be transferred easily between SCIM aware applications.

Apache SCIMple updates to Jakarta Packagesposted on Jun 1, 2022

Apache SCIMple is compatible with Jakarta EE 9. All javax packages have been replaced with the jakarta equivalent.

Projects migrating from PennState/SCIMple-Identity may need to update their web servers.
