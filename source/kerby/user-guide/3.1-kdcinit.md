# 3.1 - kdcinit

### SYNOPSIS

kdcinit [conf_dir] [keytab]

### DESCRIPTION

kdcinit is used by admin to initialize and prepare all kinds of KDC side materials, like initializing concrete back end, 
setting necessary principals (tgs, kadmin) and etc.
