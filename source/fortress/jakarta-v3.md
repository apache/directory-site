# Fortress Jakarta Compatibility

Apache Fortress 3.0 requires:

- Java 11 (core, realm)
- Java 17 (web, rest)
- Tomcat 10 (web, rest)
- Apache CXF 4 (rest)
- Apache Wicket 10 (web)
- Spring 6 (web, rest)
